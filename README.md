## Como rodar o projeto

### Pré-requisitos
- Node.js instalado.
- Expo Go instalado no seu dispositivo móvel (ou um emulador Android/iOS configurado).

### Passos
1. **Clone o repositório (opcional):**
   ```bash
   cd trabalho-react-native-g1
   ```

2. **Instale as dependências:**
   ```bash
   npm install
   ```

3. **Inicie o servidor do Expo:**
   ```bash
   npx expo start
   ```

4. **Abra o projeto:**
   - Use o app **Expo Go** para ler o QR Code exibido no terminal.
   - Ou pressione `a` para abrir no emulador Android.
   - Ou pressione `i` para abrir no simulador iOS.
   - Ou pressione `w` para abrir no navegador (web).

## Autenticação

Para realizar o login no aplicativo, você pode verificar a lista de usuários válidos (com seus respectivos `username` e `password`) através da Fake Store API:

- **Endpoint de Usuários:** [https://fakestoreapi.com/users](https://fakestoreapi.com/users)

Basta realizar um GET na URL acima para visualizar os dados de acesso.
